== Description ==

Adds extra tags to wp_head area to point between http://www.jacuzzi.com/hot-tubs/ http://www.jacuzzi.ca/hot-tubs/ and http://www.jacuzzi.ca/hot-tubs/fr/

You can find the source code at the GitHub repository:
https://github.com/ninthlink/nlk-plugins/tree/master/jhtfr-hreflang


== Installation ==

1. Upload file to the `/wp-content/plugins/` directory
2. Activate the JHTFR HREFLANG plugin through the 'Plugins' menu in the WordPress Admin Panel. Just set it, and forget it
